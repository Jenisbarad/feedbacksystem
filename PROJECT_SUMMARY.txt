================================================================================
API ENDPOINTS
================================================================================

1. POST /api/feedback
   - Submit new feedback
   - Body: { name, email, message, rating }
   - Returns: Created feedback object

2. GET /api/feedback
   - Get all feedbacks
   - Returns: Array of feedback objects
   - Sorted by createdAt DESC

3. GET /api/stats
   - Get analytics data
   - Returns: { total, avgRating, positive, negative }

4. GET /api/health
   - Health check endpoint
   - Returns: { status: "OK", message: "Server is running" }
