================================================================================
QUICK START GUIDE - LOCAL DEVELOPMENT
================================================================================

This guide will help you run the feedback form website on your local machine.

================================================================================
STEP 1: INSTALL DEPENDENCIES
================================================================================

1.1. Install Backend Dependencies:
    - Open terminal/command prompt
    - Navigate to server directory: cd server
    - Run: npm install
    - Wait for installation to complete

1.2. Install Frontend Dependencies:
    - Open a NEW terminal/command prompt window
    - Navigate to client directory: cd client
    - Run: npm install
    - Wait for installation to complete

================================================================================
STEP 2: START THE BACKEND SERVER
================================================================================

2.1. In the server terminal:
    - Make sure you're in the server directory
    - Run: npm start
    - You should see: "Server is running on port 5000"
    - Keep this terminal window open

================================================================================
STEP 3: START THE FRONTEND APPLICATION
================================================================================

3.1. In the client terminal:
    - Make sure you're in the client directory
    - Run: npm start
    - Your browser should automatically open to http://localhost:3000
    - If not, manually navigate to http://localhost:3000

================================================================================
STEP 4: TEST THE APPLICATION
================================================================================

4.1. Fill out the feedback form:
    - Enter your name
    - Enter your email
    - Enter a message
    - Click "Submit Feedback"

4.2. Verify:
    - You should see a success message
    - The form should clear
    - Check server/data/feedback.json to see stored feedback

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Port 5000 already in use
Solution: 
- Change PORT in server/index.js or set PORT environment variable
- Or stop the process using port 5000

Issue: Port 3000 already in use
Solution:
- React will ask to use a different port (like 3001)
- Accept the prompt or manually set PORT=3001

Issue: npm install fails
Solution:
- Make sure Node.js is installed (node --version)
- Try deleting node_modules and package-lock.json, then npm install again
- Check your internet connection

Issue: CORS errors in browser console
Solution:
- Make sure backend is running on port 5000
- Check that CORS is enabled in server/index.js (it should be)

================================================================================
STOPPING THE APPLICATION
================================================================================

- Press Ctrl+C in both terminal windows to stop the servers
- Close the terminal windows
